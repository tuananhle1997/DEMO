css display: inline-block

inline-block là sự kết hợp inline block

chú ý: các thẻ inline như span, a, img thì đặt width height trong css không có tác dụng

con lai của block là inline

nó sẽ lấy đặc tính hiển thị trên 1 hàng của inline

nó sẽ lấy đặc tính đặt được width và height trong css của block

khi nào cần sử dụng display:inline block

cần đặt width và height choc ác thẻ inline mà khồn muốn làm mất đặc tính hiển thị trên 1 hàng của các thẻ inline như span, a,img
