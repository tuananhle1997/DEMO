HTML ID

ID chỉ ra 1 sử duy nhất không trùng lặp không có cái thứ 2

giống như số chứng minh thư của 1 người

Giá trị id có thể được sử dụng bở css hay js để thực hiện 1 nhiệm vụ duy nhất cho 1 thành phần với 1 giá trị cụ thể

Trong css để chọn 1 id duy nhất của thẻ html ta sử dụng selector là #id

Vẫn ví dụ trên cần css cho thủ đô hà nội 1 sự đặc biệt vì đó là thủ đô duy nhất

Thêm cho thẻ div chứa hà nội 1 id tên là main
