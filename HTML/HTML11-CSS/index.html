CSS là viết tắt của Cascading Style Sheets

CSS mô tả cách các thành phần HTML được hiển thị trên màn hình và trên các thiết bị khác.

CSS tiết kiệm rất nhiều thời gian làm việc

Nó có thể kiểm soát bố cục của rất nhiều trang web cùng 1 thời điểm

CSS được thêm vào HTML theo 3 cách

Cách 1 gọi là inline css ( sử dụng thuộc tính style="" trong thẻ html)

Cách 2 gọi là internal css ( sử dụng thẻ style trong thẻ head )

Cách 3 gọi là external css ( sử dụng 1 file css bên ngoài file html )
